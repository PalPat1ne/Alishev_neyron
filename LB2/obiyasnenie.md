# Объяснение кода `lb2.cpp`

Ниже идет разбор программы `lb2.cpp` для лабораторной работы №2.

---

## Что делает программа

Программа моделирует обучение нейрона с учителем.

Есть:

- эталонный нейрон с весами `w0`
- обучаемый нейрон с весами `wL`

Задача программы:

1. считать эталонные коэффициенты из файла
2. генерировать случайные входные сигналы
3. пропускать их через эталонный и обучаемый нейрон
4. изменять веса обучаемого нейрона
5. смотреть, насколько веса `wL` приблизились к `w0`

---

## Подключение библиотек

```cpp
#include <iostream>
#include <fstream>
#include <ctime>
#include <cmath>
```

Что используется:

- `iostream` для `cout` и `cerr`
- `fstream` для чтения и записи файлов
- `ctime` для `time(NULL)`
- `cmath` для `tanh()`

---

## Функция активации

Код:

```cpp
float f(float x) { return tanh(x); }
```

Эта функция возвращает:

```text
y = tanh(x)
```

То есть в программе используется биполярная сигмоидальная функция активации.

Она нужна, чтобы из суммы входов получить выход нейрона.

---

## Функция шума

Код:

```cpp
float noise(float s)
{
    long long r;
    int i;
    for (r = -(RAND_MAX + 1LL) * 6, i = 0; i < 12; i++)
        r += rand();
    s *= r / (RAND_MAX + 1.);
    return s;
}
```

Что делает эта функция:

- несколько раз вызывает `rand()`
- суммирует случайные числа
- умножает результат на `s`

По смыслу это добавка случайного шума к входному сигналу.

Параметр:

- `s` это уровень шума, то есть `sigma`

---

## Точка входа в программу

Код:

```cpp
int main(int argn, char *argv[])
```

Программа принимает аргументы командной строки.

Запуск выглядит так:

```bash
./lb2 coef.txt Niter gamma sigma
```

или так:

```bash
./lb2 coef.txt Niter gamma sigma eps_history.txt
```

Где:

- `coef.txt` это файл эталонных коэффициентов
- `Niter` это число шагов обучения
- `gamma` это скорость обучения
- `sigma` это уровень шума
- `eps_history.txt` это файл для записи истории ошибки

---

## Основные переменные

Код:

```cpp
int i, k, K, N;
float s0, sL, y0, yL, t, gamma = 0.1, sigma = 0, e2 = 0, min_eps = 0;
const float eps_limit = 0.1f;
```

Что означает:

- `K` это число коэффициентов
- `N` это число шагов обучения
- `s0` это сумма для эталонного нейрона
- `sL` это сумма для обучаемого нейрона
- `y0` это выход эталонного нейрона
- `yL` это выход обучаемого нейрона
- `t` это разность выходов
- `gamma` это скорость обучения
- `sigma` это шум
- `e2` это текущая ошибка `eps2`
- `min_eps` это минимальная найденная ошибка
- `eps_limit = 0.1` это порог успешного обучения

---

## Проверка аргументов

Код:

```cpp
if (argn != 5 && argn != 6)
    cerr << argv[0] << " coef.txt Niter gamma sigma [eps_history.txt]" << endl, exit(1);
```

Программа проверяет, что ей передали нужное число аргументов.

---

## Открытие файла коэффициентов

Код:

```cpp
ifstream in(argv[1]);
if (!in)
    cerr << "No read input file \"" << argv[1] << "\"" << endl, exit(2);
```

Здесь открывается файл коэффициентов.

Если файл не найден, программа завершится с ошибкой.

---

## Подсчет количества коэффициентов

Код:

```cpp
for (K = 0;;)
    if (in >> s0)
        K++;
    else
        break;
```

Что делает этот цикл:

- читает числа из файла
- считает, сколько коэффициентов записано

После этого:

- `K` показывает число весов нейрона

---

## Чтение параметров из командной строки

Код:

```cpp
N = atoi(argv[2]);
gamma = atof(argv[3]);
sigma = atof(argv[4]);
```

Что здесь происходит:

- строка `argv[2]` переводится в число шагов
- строка `argv[3]` переводится в `gamma`
- строка `argv[4]` переводится в `sigma`

---

## Создание массивов весов

Код:

```cpp
float *w0 = new float[K];
float *wL = new float[K];
```

Здесь:

- `w0` хранит веса эталонного нейрона
- `wL` хранит веса обучаемого нейрона

Сначала `wL` заполняется нулями.

---

## Чтение эталонных весов

Код:

```cpp
in.open(argv[1]);
srand(time(NULL));
for (i = -1; ++i < K; in >> w0[i], wL[i] = 0)
    ;
```

Что делает этот блок:

- снова открывает файл коэффициентов
- записывает значения в массив `w0`
- одновременно обнуляет `wL`
- запускает генератор случайных чисел `srand(time(NULL))`

---

## Создание массива входных сигналов

Код:

```cpp
float *x = new float[K];
```

Массив `x` хранит текущий входной сигнал для каждого коэффициента.

---

## Запись истории ошибки в файл

Код:

```cpp
ofstream out;
if (argn == 6)
    out.open(argv[5]);
```

Если при запуске передан шестой аргумент, программа открывает файл для записи истории `eps2`.

В этот файл потом пишутся:

- номер шага
- значение ошибки

---

## Начальная ошибка

Код:

```cpp
for (e2 = 0, i = -1; ++i < K;)
    e2 += (w0[i] - wL[i]) * (w0[i] - wL[i]);
min_eps = e2;
if (argn == 6)
    out << 0 << '\t' << e2 << endl;
```

Здесь считается начальная ошибка между эталонными и обучаемыми весами.

Формула:

```text
eps2 = ||w0 - wL||^2
```

То есть суммируются квадраты разностей всех коэффициентов.

Шаг `0` записывается в файл как состояние до обучения.

---

## Основной цикл обучения

Код:

```cpp
for (k = -1; ++k < N;)
{
    ...
}
```

Этот цикл выполняется `N` раз.

На каждом шаге программа:

1. генерирует входные сигналы
2. считает выход эталонного нейрона
3. считает выход обучаемого нейрона
4. вычисляет ошибку
5. меняет веса `wL`
6. пересчитывает `eps2`

---

## Генерация входных сигналов

Код:

```cpp
for (i = -1; ++i < K;)
    x[i] = 1 - 2 * (rand() % 2) + noise(sigma);
```

Что происходит:

- `rand() % 2` дает `0` или `1`
- выражение `1 - 2 * (rand() % 2)` дает `1` или `-1`
- затем добавляется шум `noise(sigma)`

То есть входные сигналы случайные и могут быть:

```text
-1 + шум
или
1 + шум
```

---

## Работа эталонного нейрона

Код:

```cpp
for (s0 = sL = 0, i = -1; ++i < K;)
    s0 += x[i] * w0[i];
y0 = f(s0);
```

Здесь:

- считается сумма входов и эталонных весов
- потом к сумме применяется функция `tanh`

Это выход эталонного нейрона.

---

## Работа обучаемого нейрона

Код:

```cpp
for (sL = 0, i = -1; ++i < K;)
    sL += x[i] * wL[i];
yL = f(sL);
```

То же самое делается для обучаемого нейрона, только используются веса `wL`.

---

## Ошибка выходов

Код:

```cpp
t = y0 - yL;
```

Здесь находится разность выходов:

- `y0` — правильный ответ эталонного нейрона
- `yL` — ответ обучаемого нейрона

Чем меньше `t`, тем ближе обучаемый нейрон к эталонному.

---

## Изменение весов

Код:

```cpp
for (i = 0; i < K; i++)
    wL[i] += gamma * x[i] * t;
```

Это главное правило обучения.

Здесь:

- `gamma` задает скорость обучения
- `x[i]` это текущий вход
- `t` это ошибка

Если ошибка большая, веса меняются сильнее.

---

## Пересчет `eps2`

Код:

```cpp
for (e2 = 0, i = -1; ++i < K;)
    e2 += (w0[i] - wL[i]) * (w0[i] - wL[i]);
```

После изменения весов программа снова считает ошибку между `w0` и `wL`.

Потом:

```cpp
if (e2 < min_eps)
    min_eps = e2;
```

Так программа запоминает наименьшее значение ошибки за все время обучения.

---

## Запись `eps2` по шагам

Код:

```cpp
if (argn == 6)
    out << k + 1 << '\t' << e2 << endl;
```

Если открыт файл истории, программа записывает в него:

- номер шага
- значение `eps2`

Это потом удобно использовать для графиков.

---

## Поиск первого успешного шага

Код:

```cpp
if (success_step < 0 && e2 < eps_limit)
    success_step = k + 1;
```

Если ошибка впервые стала меньше `0.1`, программа запоминает номер этого шага.

---

## Итоговый вывод

После цикла программа выводит:

- разность выходов `y0 - yL`
- эталонные веса `w0`
- обученные веса `wL`
- финальное значение `eps2`
- минимальное значение `eps2`
- первый шаг, на котором ошибка стала меньше `0.1`

---

## Закрытие файла и освобождение памяти

Код:

```cpp
if (argn == 6)
    out.close();
delete[] x;
delete[] wL;
delete[] w0;
```

Здесь:

- закрывается файл истории
- освобождается выделенная память

---

## Пример запуска

Сборка:

```bash
clang++ -std=c++17 LB2/lb2.cpp -o LB2/lb2
```

Запуск без истории:

```bash
./LB2/lb2 LB2/coef6.txt 1000 0.1 0.5
```

Запуск с историей:

```bash
./LB2/lb2 LB2/coef6.txt 1000 0.1 0.5 LB2/eps_history.txt
```

---

## Короткий вывод

Программа `lb2.cpp` обучает нейрон с учителем.  
Она сравнивает эталонные веса и обучаемые веса, постепенно изменяет `wL` и считает ошибку `eps2`.  
Если ошибка становится меньше `0.1`, считается, что обучение прошло успешно.
