# Что было дописано в `lb2.cpp`

Этот файл объясняет только те части, которые были дописаны в код для второй лабораторной работы.

---

## 1. Добавлен расчет `eps2`

В программе добавлен расчет ошибки между эталонными и обученными весами:

```text
eps2 = ||w0 - wL||^2
```

То есть программа считает сумму квадратов разностей всех коэффициентов.

В коде это сделано обычным циклом:

```cpp
for(e2=0,i=-1;++i<K;) e2+=(w0[i]-wL[i])*(w0[i]-wL[i]);
```

Это значение нужно для контроля, насколько обучаемый нейрон приблизился к эталонному.

---

## 2. Исправлено использование шума

В варианте с фото было:

```cpp
x[i]=1-2*(rand()%2)+noise(1);
```

Сейчас:

```cpp
x[i]=1-2*(rand()%2)+noise(sigma);
```

Это сделано потому, что в лабораторной работе надо исследовать влияние `sigma`.

Если оставить `noise(1)`, то параметр `sigma` из командной строки не будет влиять на результат.

---

## 3. Добавлена запись `eps2` в файл

Программа может принимать дополнительный аргумент:

```bash
./LB2/lb2 coef.txt 1000 0.1 0.5 eps_history.txt
```

Если этот аргумент указан, открывается файл:

```cpp
ofstream out;
if(argn==6) out.open(argv[5]);
```

Дальше на каждом шаге в файл записываются:

- номер шага
- текущее значение `eps2`

Пример строки:

```text
25    0.4382
```

То есть теперь не нужно хранить историю в памяти, программа сразу пишет данные в файл.

---

## 4. Добавлена запись начального значения

Перед началом обучения программа считает начальную ошибку и тоже записывает ее в файл:

```cpp
for(e2=0,i=-1;++i<K;) e2+=(w0[i]-wL[i])*(w0[i]-wL[i]);
if(argn==6) out<<0<<'\t'<<e2<<endl;
```

Шаг `0` означает состояние до обучения.

Это полезно для графика, потому что видно, с какого значения все началось.

---

## 5. Добавлен поиск первого успешного шага

В коде добавлена простая проверка:

```cpp
if(success_step<0 && e2<eps_limit) success_step=k+1;
```

Где:

```cpp
const float eps_limit=0.1f;
```

Это значит:

- если ошибка впервые стала меньше `0.1`
- программа запоминает номер этого шага

Если такого шага не было, выводится:

```text
first_success_step=not_found
```

---

## 6. Добавлено минимальное значение ошибки

Во время обучения программа еще запоминает минимальное значение `eps2`:

```cpp
if(e2<min_eps) min_eps=e2;
```

В конце это выводится на экран:

```text
min_eps2=...
```

Это нужно, чтобы потом было проще смотреть результат эксперимента.

---

## 7. Убрано автоматическое определение вариантов ошибки

Раньше в коде была более сложная логика:

- хранение всей истории в `vector`
- автоматическое определение вариантов `A`, `B`, `V`

Сейчас это убрано.

Почему:

- код стал проще
- данные по `eps2` и так пишутся в файл
- по этим данным можно самому построить график и определить поведение ошибки

То есть программа теперь просто считает и сохраняет результаты, без лишней автоматической аналитики.

---

## 8. Что выводит программа теперь

После завершения программа выводит:

- исходные веса `w0`
- полученные веса `wL`
- финальное значение `eps2`
- минимальное значение `eps2`
- первый шаг, где ошибка стала меньше `0.1`

Это выглядит примерно так:

```text
||w0-wL||^2=0.099
min_eps2=0.099
first_success_step=123
```

или

```text
||w0-wL||^2=0.352
min_eps2=0.214
first_success_step=not_found
```
